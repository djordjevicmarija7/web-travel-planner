# Travel Planner

Web aplikacija za planiranje putovanja.

## Tehnologije

- Frontend: React + Vite
- Backend: Microsoft Service Fabric (.NET 7)
- Baza: SQL Server

## Pokretanje frontend aplikacije

1. Instaliraj zavisnosti:
   npm install

2. Napravi .env fajl (vidi .env.example):
   cp .env.example .env

3. Pokreni aplikaciju:
   npm run dev

Aplikacija je dostupna na http://localhost:5173
